1. What is a Dataset?

A dataset is a structured collection of data, typically organized in rows and columns. Each row represents a statistical unit (e.g., a person, an object, an event), and each column represents a variable or feature describing the unit (e.g., age, gender, score).

2. What is a Statistical Distribution?

A statistical distribution describes how the values of a variable are spread across the dataset. It shows the frequency or probability of each possible value or range of values. Distributions help us understand the central tendency, variability, and shape of the data.
