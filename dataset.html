Explain the concept of Dataset and Distribution.

Dataset

A **Dataset** is a structured collection of data. It is typically organized in a tabular format (like a database table), where **Rows** represent the **Statistical Units** (observations) and **Columns** represent the **Variables** (features) being measured.

Distribution

A **Statistical Distribution** describes how the values of a variable are **spread out** across the entire dataset. It is a way of summarizing the data to show which values are possible and how often (with what frequency) each value appears.
